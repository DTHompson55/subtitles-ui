Subtitler

This Subtitling Service has 5 steps

Step1 - Submit a video file for processing

Step2 - Verify correct file and make transcription selections

Step3 - Wait for Watson to transcribe the file

Step4 - Review results and make any corrections

Step5 - Submit Closed Caption file to Streaming Service

Please Note: There is an approximately 10 minute limit for video files in this demo.
